What's already live: hero parallax · marquee · stamp rotation · map pin · service card v1c · gallery tile hover · button hovers · nav underline · scroll reveal + stagger · backdrop modal.

What this companion proposes adding:

  • Pricing row hover (subtle bg + indent) — adds scan affordance to the price list
  • Eyebrow dash draw — replaces static "— " with a 22px line that scales in on reveal
  • Section divider draw — between major sections, subtle but signals hierarchy
  • Refined timing on existing hovers — pulled into the same 200/400/600/750ms scale
  • Modal scale-in choreography — backdrop blur + content scale 0.96 → 1, current modal is fade-only

What this companion explicitly skips: magnetic cursor effects, tilt-on-hover, glow trails, particles, scroll-jacking, page-load loaders, bounce easing on H1/H2, anything that would feel SaaS instead of editorial.
