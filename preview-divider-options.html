Recommendation.

My pick: Option C — Partner Credentials. It's the only option that adds information the page didn't already have. Stats (A) duplicate hero-meta. Chapter divider (B) is pure decoration. Map (D) duplicates Visit. The implant brand list belongs somewhere on the page — currently it's only in the Implantology service card body, which is small. Surfacing it as a band gives the credibility signal real estate.

Strong second: Option B — Chapter Divider if you specifically want editorial restraint and don't mind the "decoration only" framing. Smallest functional ask, biggest visual ROI.

Skip A: the hero-meta strip (9 lékařů · 4 dentální hygienistky) already does this work, just smaller. Repeating it as a band is redundant.

Skip D: the Visit section with the live MapLibre map already lives below — having a faded map preview earlier weakens the live one's payoff.

If you want functional credibility Pick C — Partners
If you want pure editorial restraint Pick B — Chapter Divider
If you want trust numbers Pick A — Stats (but it duplicates hero-meta)
If you want spatial anchor Pick D — Map (but it weakens Visit)
