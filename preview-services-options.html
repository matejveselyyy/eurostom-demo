Services design — 4 directions

Beyond the 3-card grid.

The current section is six identical cards in a 3-column grid. It works, but it's the most generic editorial move possible. Below: four directions that lean into different design conventions — magazine TOC, editorial index with detail panel, numbered stack, asymmetric featured grid. Same six services, same Czech copy. Pick the direction that feels right.

Option A · Editorial Index

Rail navigation + detail panel.

Six services on the left as a numbered editorial list. Click any one to load its detail in the right panel — full description, large background numeral, meta data. Magazine table-of-contents meets product detail page. Most refined option for an editorial luxury brand.

Option B · Numbered Stack

Single column, big numerals, full width.

All six services as horizontal rows. Massive numerals on the left, service name + tag in the middle, description on the right. Hover slides the row 16px right and shifts the numeral up. Reads like a printed index in a serious medical journal — but warmer.

Option D · Magazine TOC

Like the contents page of a serious magazine.

Centered editorial heading. Six services as horizontal lines, each row: number column + name + short description + arrow. Hover draws an accent line and reveals the arrow. Very Aesop / Apartamento / The Plant magazine. Quietest of the four options — pure typography.

Recommendation.

For maximum editorial luxury feel, my pick is Option D — Magazine TOC. It's the quietest, the most "Maison Abelane / Aesop" of the four. Pure typography, no card backgrounds, generous whitespace. The clinic's services don't need to be sold like SaaS features — they need to be presented like a curated index.

Second pick: Option A — Editorial Index. Adds interactivity (rail + detail) without losing the editorial restraint. Best if you want users to actively explore each service rather than scan all six in one glance.

Skip: Option C unless you specifically want to lead with Implantology (highest-revenue service). The current 3×2 grid is essentially a quieter version of C, so swapping isn't a huge upgrade.

If you want pure restraint Pick D — Magazine TOC
If you want interactivity Pick A — Editorial Index
If you want full-width drama Pick B — Numbered Stack
If you want to feature implants Pick C — Featured + Grid
